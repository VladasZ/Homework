Конструкторы класса Справочник:

Конструктор по умолчанию - создает массив из 20-ти случайно генерируемых компаний.
Конструктор принимающий string - принимает адрес файла из которого будет создан новый справочник.


	Реализованные методы:
		
		- Отображение всех компаний на экран
		- Сохранение справочника в файл. Принимает string. Сохраняет адрес файла.
		- Добавление новой компании:
			- принимает 4 поля string с данными об компании
			- если справочник уже был сохранен, добавляет новую компанию в существующий файл.

При создании новой базы, автомобили и нарушения генерируются случайно. 
По умолчанию 20 машин, есть конструктор принимающий int - нужное количество генерируемых машин.
Структура данных - двоичное дерево.

 Реализованные функции:

	- Добавление новой машины:
		параметры: номер и нарушение, если такой номер уже существует
		нарушение добавляется к существующей машине

	- Отображение машины:
		можно вызвать для каждой отдельной машины
		показывает номер и все нарушения

	- Перегружен оператор [] для доступа к любой машине по номеру
		если такого номера нет то выводит сообщение что машина не найдена

	- Функция вывода на экран всех машин в отсортированном виде
	
	- Функция вывода на экран всех машин в заданном диапазоне

	- Функция сохранения всей базы в тектовой файл. Принимает адрес файла.

	- Функция сохранения диапазона номеров в файл. 
		принимает адрес файла и 2 int - диапазон.
